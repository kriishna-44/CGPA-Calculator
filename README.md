# 🎓 CGPA Calculator

A modern CGPA Calculator built using HTML, CSS, and JavaScript.

## Features

- Calculate CGPA for up to 8 semesters
- Target CGPA Planner
- Highest SGPA Tracker
- Lowest SGPA Tracker
- Average SGPA Analysis
- Responsive Design
- Modern Glassmorphism UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

1. Select the number of semesters.
2. Enter SGPA values.
3. Click "Calculate CGPA".
4. Use the Target CGPA Planner to determine the SGPA needed in remaining semesters.
